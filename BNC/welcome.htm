What's in this package

  • BNC XML Edition: the corpus
    The BNC is a 100 million word snapshot of British English at the end of the twentieth century, made up of over 4000 distinct "texts", spoken and written, sampled from a wide range of different kinds of writing. The XML Edition is an enhanced and revised version of the original BNC World Edition, containing additional linguistic annotation, and many corrections.
  • The BNC Users Reference Guide
    This detailed manual describes the corpus design and contents, with full reference information about its encoding scheme, and suggestions for how to process and manage it with freely available standard software tools. Any XML aware software can be used to access the XML files making up the corpus, once it has been unpacked.

>What's not in this package

  • Xaira
    The BNC XML package was previously supplied on disks with a set of pre-built Xaira indexes which enable you to start searching the corpus straight away with Xaira, a powerful open source XML indexing and searching software system, designed especially for use with the BNC. However, the University of Oxford, who distributes the corpus, is not able to support Xaira, nor ensure that the latest version of the software and indexed data are available. You can find out more about Xaira, including software downloads, from http://www.xaira.org.
